## Style Guide for All Screenshots

| Property | Value |
|---|---|
| Background | Light gradient #f0f3ff to #f8fafc |
| Headlines | Inter Semibold 40px, color #1e293b |
| Subtext | Inter Regular 20px, color #475569 |
| Extension UI | Realistic size, subtle shadow `0 20px 40px rgba(0,0,0,0.15)` |
| Branding | Small Zovo logo bottom-right, 60-80% opacity, ~80px wide |

### Do NOT

- Use placeholder text (lorem ipsum, etc.)
- Show empty states
- Include personal data (real emails, names)
- Clutter backgrounds with unnecessary elements
