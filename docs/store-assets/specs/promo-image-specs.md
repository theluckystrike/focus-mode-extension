
| Property | Value |
|---|---|
| Dimensions | 1400x560 PNG |
| Background | Gradient #6366f1 to #4f46e5 (left to right) |
| Left side | Icon + extension name + tagline + 3 bullet features |
| Right side | Popup UI preview mockup with realistic data |
| Title text | White, Inter Bold, "FOCUS MODE - BLOCKER" |
| Tagline text | White, Inter Regular |
| Feature bullets | White, Inter Regular, star/bullet prefix |
| Zovo logo | Bottom right, white, small (~100px wide) |

### Left Side Details

- Shield icon: White, 160-200px, vertically centered with text
- Title: Inter Bold 48px, all caps
- Tagline: Inter Regular 24px, three lines stacked
- Spacing: 16px between tagline and bullet features
- Bullet features: Inter Regular 20px, each with a star prefix

### Right Side Details

- Popup UI mockup: Floating with subtle shadow `0 20px 60px rgba(0,0,0,0.3)`
- Rounded corners: 12px
- Show realistic popup content: blocklist with sites, Focus Score ring, timer at 25:00
- Slight rotation: 2-3 degrees clockwise for visual interest (optional)
- Scale: Popup should be roughly 320px wide in the mockup

## File Naming Convention

When exporting final assets, use these filenames:

| Asset | Filename |
|---|---|
| Small promo tile | `promo-small-440x280.png` |
| Marquee promo tile | `promo-marquee-1400x560.png` |
| Social share image | `social-share-1200x630.png` |

All files should be exported as PNG with no transparency (solid backgrounds).
