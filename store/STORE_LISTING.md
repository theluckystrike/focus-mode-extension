## Permission Justifications

### storage
```
Used to save user preferences, blocklist rules, whitelist rules, and focus statistics locally on the device. No data is transmitted externally.
```

### tabs
```
Required to detect when users navigate to blocked sites and redirect them to the blocked page. Tab information is processed locally and never transmitted externally.
```

### webNavigation
```
Required to intercept navigation to blocked websites before the page loads, providing instant blocking without loading the distracting content first.
```

### alarms
```
Used to manage focus session timers, break reminders, and scheduled focus mode activation. Runs entirely locally with no external communication.
```

### notifications
```
Used to notify users when focus sessions complete, breaks end, or when it's time to take a break. All notifications are generated locally.
```

### host_permissions (<all_urls>)
```
Required to block distracting websites on any domain the user configures. Only activates during focus mode and only for sites on the user's blocklist.
```
