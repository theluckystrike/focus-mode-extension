COMMENT DEMARRER :
1. Installez Focus Mode Pro depuis le Chrome Web Store
2. Cliquez sur l'icône dans la barre d'outils pour ouvrir le panneau
3. Choisissez le mode Pomodoro, Personnalisé ou Indéfini
4. Appuyez sur Démarrer la concentration — les sites distrayants sont bloqués automatiquement !

RACCOURCI CLAVIER :
